# Amemori Labs

**Amemori Labs** is a technology company building software tools, APIs, and data analytics products.

## About

Founded in 2026 and based in London, Amemori Labs was established by alumni of EPFL and Imperial College London. The company focuses on research and development in software and applied machine learning.

## Mission

Our mission is to make complex tools more efficient and intuitive to use.

## Philosophy

Amemori Labs is inspired by innovation, combining an Apple-like design philosophy with a strong emphasis on simplicity and usability.

## What We Do

- Software tools  
- APIs  
- Data analytics products  
- Applied machine learning solutions  

## Audience

We build tools for:
- Businesses  
- Individuals
